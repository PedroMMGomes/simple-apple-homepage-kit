Site institucional de software house com **backend próprio** (`backend/`) + Supabase. Vite + React + TypeScript + Tailwind + shadcn/ui.

## 🚀 Quick start

```bash
npm install
cp .env.example .env     # chaves públicas do Supabase
npm run dev
npm run build
```

> As chaves em `.env` são **públicas por design** (anon do Supabase, protegidas por RLS). `.env` fica fora do git.
